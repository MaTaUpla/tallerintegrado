Acerca de la web
    ¿Que es esto?
  • Es una web app para sacar informacion de los streamers conocidos o poco conocidos de la plataforma Twitch.tv solo se necesita su username para obtener la informacion, esta esta conectada a la api de twitch reflejando datos reales de los individuos
    ¿La informacion de donde la saco?
  • La informacion sacada es de la api oficial de Twitch la cual es abierta para todo publico con fines educativos, estadisticos o para crear herramientas de stream, al ser de oficial de twitch los datos son actualizados cada x tiempo y no son modificables
    ¿Quien la diseño?
  • La WebApp fue diseñada por Matias Tapia alumno de quinto año de Ingenieria Informatica con fines de obtener informacion y aplicar el uso de una api con un framewrok tipo MVC
